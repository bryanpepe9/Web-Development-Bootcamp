About Me

I am Bryan C. Pepe, a full-stack developer with a Bachelor of Science in Computer Science from Florida International University. With a strong foundation in computer science principles and coursework covering a wide range of topics, I possess advanced proficiency in Python, HTML, CSS, and SQL, as well as intermediate knowledge in Java, JavaScript, and C. Throughout my academic journey and professional experience, I have developed software applications in multiple languages, conducted data analysis using libraries like Pandas and Numpy, and utilized automation tools such as Selenium and Beautiful Soup to enhance efficiency. I have worked on web development projects, where I leveraged frameworks like Flask and Bootstrap to create user-friendly applications. Additionally, I have gained experience in database management, RESTful API testing with Postman, and creating efficient and scalable databases using MySQL and SQLAlchemy. My goal is to utilize my diverse skill set, along with my passion for continuous learning, to create innovative and impactful applications that optimize functionality and user experience.
